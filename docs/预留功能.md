# 预留功能记录

## 赛道管理高级界面
- `app/src/main/java/com/example/racerstats/track/ui/TrackListFragment.kt`：计划承载赛道列表、支持新增/选择/长按操作。
- `app/src/main/java/com/example/racerstats/track/ui/TrackDetailFragment.kt`：展示单条赛道的地图、统计信息与操作入口。
- `app/src/main/java/com/example/racerstats/track/ui/TrackEditFragment.kt`：用于修改赛道名称、描述、起终点等元数据。
- `app/src/main/java/com/example/racerstats/track/ui/TrackListAdapter.kt`：`RecyclerView` 适配器，负责赛道列表项绑定与点击回调。
- `app/src/main/res/layout/fragment_track_list.xml`：赛道列表页面布局，包含列表、空态提示与添加按钮。
- `app/src/main/res/layout/fragment_track_detail.xml`：赛道详情布局，预留地图展示、统计矩阵与操作条。
- `app/src/main/res/layout/fragment_track_edit.xml`：赛道编辑布局，含输入表单、验证提示与保存/取消控件。
- `app/src/main/res/menu/track_item_menu.xml`：赛道列表项右上角溢出菜单，预期提供编辑、删除、导出。
- `app/src/main/res/menu/track_detail_menu.xml`：详情页工具栏菜单，准备挂载编辑、分享、更多操作。
- `app/src/main/res/menu/track_edit_menu.xml`：编辑页工具栏菜单，承载保存/撤销等动作。
- `app/src/main/res/navigation/nav_graph.xml`：导航图，定义赛道列表 → 详情 → 编辑的跳转流程。

> 以上页面与资源未在 `MainActivity` 中接入导航流程，仅在 `NavHostFragment` 场景下可用。若未来要启用完整赛道列表/详情/编辑功能，可复用这些文件。

## 备用图标资源
- `app/src/main/res/drawable/ic_check.xml`：计划用于赛道录制完成或校验成功的高亮状态，当前未挂载。
- `app/src/main/res/drawable/ic_close.xml`：预留给对话框/菜单关闭按钮，暂未在 UI 中引用。
- `app/src/main/res/drawable/ic_file.xml`：原设计用于赛道文件/导入入口，功能未上线。
- `app/src/main/res/drawable/ic_info.xml`：预计在赛道详情说明或提示面板显示，尚无引用场景。
