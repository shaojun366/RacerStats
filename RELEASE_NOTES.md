# RacerStats v1.0.1 Release Notes

## 🎉 新版本发布 - 赛道构建功能初步实现

### 📅 发布信息
- **版本号**: v1.0.1
- **发布日期**: 2025年11月12日
- **APK 大小**: ~8MB
- **最低支持**: Android 7.0 (API 24)

### ✨ 主要新功能

#### 🎯 **赛道宽容度调节系统**
- **四种预设模式**: 精密赛道(25m) | 街道赛道(40m) | 一般道路(50m) | 高速公路(80m)
- **一键切换**: 录制界面实时调节匹配精度
- **智能适配**: 根据不同驾驶环境自动优化轨迹匹配
- **实时反馈**: 选择后立即生效，按钮显示当前设置

#### 🔧 **UI/UX 改进**
- **优化布局**: 重构录制界面统计信息区域
- **水平排列**: 距离、点数、宽容度信息一目了然
- **Material Design**: 遵循现代 Android 设计规范
- **响应式交互**: 流畅的选择对话框和状态更新

#### 📚 **完善文档系统**
- **功能说明**: 详细的宽容度配置指南
- **使用教程**: 分步骤的操作说明
- **技术文档**: 开发者友好的实现细节
- **项目 README**: 专业的 GitHub 项目展示

### 🛠️ **技术改进**

#### **架构优化**
- **StateFlow 管理**: 现代化的状态管理方案
- **MVVM 架构**: 清晰的职责分离和数据流
- **Hilt 依赖注入**: 高效的组件管理
- **Room 数据库**: 可靠的本地数据存储

#### **核心算法**
- **轨迹匹配**: Douglas-Peucker 简化 + Hausdorff 距离
- **精度控制**: 可配置的距离阈值判断
- **GPS 平滑**: 0.7/0.3 权重滤波消除抖动
- **实时计算**: 高效的圈速和 Delta 算法

### 📊 **性能指标**
- **GPS 采样率**: 10Hz 高频数据采集
- **响应延迟**: <100ms 界面更新
- **内存优化**: 合理的资源管理
- **电量控制**: 智能的定位服务管理

### 🐛 **问题修复**
- 修复 Track 页面按钮可见性问题
- 优化地图显示和交互逻辑
- 改进 GPS 权限处理流程
- 提升应用启动和切换性能

### 📱 **安装说明**
1. 下载 `RacerStats-v1.0.1-debug.apk`
2. 在 Android 设备上允许"未知来源"安装
3. 点击 APK 文件进行安装
4. 授予定位权限以使用完整功能

### ⚠️ **注意事项**
- **Debug 版本**: 此为调试版本，包含开发者工具
- **权限要求**: 需要精确定位权限
- **Google Maps**: 确保设备有 Google 服务支持
- **存储空间**: 建议保留至少 50MB 可用空间
